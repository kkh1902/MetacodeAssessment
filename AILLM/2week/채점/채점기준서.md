- **총 10문제 × 10점 = 100점**, 부분 점수 없음 (O/X 이진 채점)
- 문제지 요구사항 기반 판정, 정답지는 참조용
- 출력 결과가 동일하고 요구사항이 모두 이행되면 다양한 구현 방식 허용

## Q1. 데이터 집계 (pivot_table)

**문제지 요구사항**
- `pivot_table` 함수 사용
- `index='date'`, `columns='source'`, `values='clicks'`, `aggfunc='mean'`
- 상위 5개만 출력 (`head(5)`)

**O 기준**: 위 조건 모두 충족 + 상위 5개 결과 출력됨
**X 사례**: pivot_table 미사용 / 인자 다름 / head(5) 누락 / 출력 없음

## Q2. 데이터 전처리 (to_datetime)

**문제지 요구사항**
- `date` 컬럼을 날짜형(datetime)으로 변환
- 변환 후 정보 확인 (`info()` 출력)

**O 기준**: `pd.to_datetime()` 등으로 date 컬럼 변환 + info()에서 datetime64 확인
**X 사례**: 변환 안 함 / info() 출력 없음 / 다른 컬럼만 변환

## Q3. 데이터 집계 및 정렬 (groupby + sort)

**문제지 요구사항**
- `groupby('category')` 집계
- 결과 컬럼명 `cost_mean`, `price_mean`
- `price_mean` 기준 **내림차순** 정렬
- `./Q3_result.csv` 저장

**O 기준**: category별 cost/price 평균 집계, 컬럼명 일치, price_mean desc 정렬, CSV 저장
**X 사례**: 컬럼명 불일치 / 오름차순 / CSV 미저장 / groupby 기준 다름

## Q4. 데이터 전처리 (filter + column select)

**문제지 요구사항**
- `category == 'a'` 필터
- `id, category, price` 세 컬럼만 유지

**O 기준**: 필터 후 3개 컬럼만 선택하여 출력
**X 사례**: 필터 조건 다름 / 컬럼 누락/추가 / 출력 없음

## Q5. 데이터 시각화 bar

**문제지 요구사항**
- `x='category'`, `y='cost_mean'`
- 제목 `Category-Cost`
- Q3 결과 데이터 사용

**O 기준**: 위 조건 충족한 bar 차트 생성 및 표시
**X 사례**: 제목 누락/오기 / 축 다름 / 차트 미생성

## Q6. 데이터 전처리 및 집계 (dropna + nunique)

**문제지 요구사항**
- `dropna(how='any')`로 결측치 제거
- `campaign, source` 기준 `user_id.nunique()` 집계
- 집계 컬럼명을 `user_num`으로 rename

**O 기준**: dropna + groupby nunique + rename(user_num) 모두 적용
**X 사례**: how 옵션 다름 / nunique 아닌 count/size / 컬럼명 user_num 아님

# AILLM 12주차 채점 기준서

> **채점 원칙: 문제당 0 or 10, 부분점수 없음**

## 채점 방식
- 각 문제별 배점: **10점** (O = 10점 / X = 0점, 이진 채점)
- 총점: **100점** (10문제 × 10점)
- O (10점): 핵심 요구사항 **모두** 충족
- X (0점): 하나라도 미흡 / 미제출 / 실행 실패

---

## Q7 특례 규칙

- **Q7은 재실행하지 않는다.** (RAG + PEFT 학습 비용이 과도하게 크고, 수강생 환경마다 GPU 상황이 달라 재현 불가)
- ipynb에 **저장된 실행 결과 셀**만 보고 채점 가능하면 채점한다.
  - 학습 루프가 완주되고 최종 loss/Accuracy가 출력되어 있으면 **O**
  - OOM/KeyboardInterrupt/NameError 등으로 중간 중단되어 최종 결과가 없으면 **X**
  - 결과 셀이 전혀 없거나 판단 불가한 경우에만 `-(Q7 채점 제외)` 처리 가능
- 학생 환경 차이로 인한 성능 편차(예: Accuracy 0% 등)는 **구현 완성도만 판정**하고 성능 수치로 감점하지 않는다.

---

## 채점 시 주의사항

1. **파일명이 달라도** 동일 기능 구현 시 O 인정
2. **ipynb 출력 셀**로 실행 결과가 보존되어 있으면 캡쳐 대체 인정
3. **정답과 다른 접근이라도** 요구사항을 모두 충족하면 O
4. **수강생 제출 원본 파일은 절대 수정/삭제하지 말 것**
5. Q1은 학습 전/후 accuracy 비교가 핵심 — **학습 후 하락 시 X**
6. Q10은 목표 Accuracy(95%) 미달 시 X

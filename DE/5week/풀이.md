## 문제 3. WordCount 실행 (Spark DataFrame API)

### 문제 요약
S3 텍스트를 `spark.read.text`로 읽어 `split` + `explode` → `groupBy` + `count`로 단어 빈도 집계, 결과를 Notebook에 출력하고 필요 시 CSV로 저장.

### 정답 코드
```python
import logging, os
from pyspark.sql import SparkSession, functions as F

logging.basicConfig(level=logging.INFO)
logging.getLogger("py4j").setLevel(logging.WARNING)
logger = logging.getLogger("WordCountApp")

AWS_ACCESS_KEY = ' '
AWS_SECRET_KEY = ' '
S3_INPUT  = "s3a://<my-bucket>/wordcount/example_address.txt"
S3_OUTPUT = "s3a://<my-bucket>/wordcount/output"

try:
    spark = (
        SparkSession.builder
        .appName("wordCount")
        .master("local[*]")
        .config("fs.s3a.access.key", AWS_ACCESS_KEY)
        .config("fs.s3a.secret.key", AWS_SECRET_KEY)
        .getOrCreate()
    )
    logger.info("SparkSession 생성 성공")

    # 1) S3에서 텍스트 로딩
    df = spark.read.text(S3_INPUT)
    logger.info(f"S3 로드 성공: {S3_INPUT}")

    # 2) 단어 분리 + 공백 제거 + 집계
    words_df = (
        df.select(F.explode(F.split(F.col("value"), r"\s+")).alias("word"))
          .where(F.length(F.trim(F.col("word"))) > 0)
    )
    word_counts = (
        words_df.groupBy("word")
                .count()
                .orderBy(F.desc("count"))
    )
    word_counts.show(20, truncate=False)

    # 3) S3에 CSV 저장 (옵션)
    word_counts.write.mode("overwrite").csv(S3_OUTPUT, header=True)
    logger.info(f"결과 저장 완료: {S3_OUTPUT}")

except Exception as e:
    logger.error(f"WordCount 실행 실패: {e}", exc_info=True)
    raise
```

### 핵심 포인트
- `SparkSession` 생성 O
- `spark.read.text(s3a://...)` 로드 O
- `split` + `explode` 단어 분리 O
- `groupBy("word").count()` 집계 O
- WordCount 결과 캡쳐본 제출 O

## 문제 4. RDD Transformation & Action

### 문제 요약
RDD 생성(`sc.textFile` 또는 `sc.parallelize`) → `flatMap` / `map` / `filter` 적용 → `reduceByKey`/`collect`로 집계 → `repartition`/`coalesce`로 파티션 조정. 각 단계 로그 기록.

### 정답 코드
```python
import logging, os
from pyspark.sql import SparkSession

logging.basicConfig(level=logging.INFO)
logging.getLogger("py4j").setLevel(logging.WARNING)
logger = logging.getLogger("RDD_S3_Transform_Action")

AWS_ACCESS_KEY = ' '
AWS_SECRET_KEY = ' '
S3_INPUT = "s3a://<my-bucket>/wordcount/example_address.txt"

try:
    spark = (
        SparkSession.builder
        .appName("rdd_wordcount")
        .master("local[*]")
        .config("fs.s3a.access.key", AWS_ACCESS_KEY)
        .config("fs.s3a.secret.key", AWS_SECRET_KEY)
        .getOrCreate()
    )
    sc = spark.sparkContext

    # 1) RDD 생성
    rdd = sc.textFile(S3_INPUT)
    logger.info(f"S3에서 RDD 생성 완료. 파일 경로: {S3_INPUT}")
    logger.info(f"초기 파티션 수: {rdd.getNumPartitions()}")

    # 2) Transformation: flatMap → filter → map
    transformed = (
        rdd.flatMap(lambda line: line.split(" "))
           .filter(lambda w: len(w.strip()) > 0)
           .map(lambda w: (w, 1))
    )
    logger.info("Transformation 단계 완료: flatMap, filter, map 수행")
    logger.info(f"Transformation 후 RDD 파티션 수: {transformed.getNumPartitions()}")

    # 3) 파티션 재분배 (repartition 또는 coalesce)
    repartitioned = transformed.repartition(4)
    logger.info(f"재분배 후 RDD 파티션 수: {repartitioned.getNumPartitions()}")

    # 4) Action: reduceByKey + collect + reduce
    word_counts = repartitioned.reduceByKey(lambda a, b: a + b).collect()
    total = sc.parallelize([c for _, c in word_counts]).reduce(lambda a, b: a + b)
    logger.info("Word Count 계산 완료")

    for w, c in word_counts:
        logger.info(f"{w}: {c}")
    logger.info(f"전체 단어 개수 합계: {total}")

    print("\n최종 Word Count 결과")
    for w, c in word_counts:
        print(f"{w}: {c}")

except Exception as e:
    logger.error(f"RDD 처리 실패: {e}", exc_info=True)
    raise
```

### 핵심 포인트
- RDD 생성(`sc.textFile`) O
- `flatMap` / `map` / `filter` 모두 사용 O
- `reduce` 또는 `reduceByKey` + `collect` O
- `repartition` (또는 `coalesce`) 파티션 조정 O
- 로그/캡쳐본 제출 O

## 문제 6. Dataset API + amount ≥ 100 필터링

### 문제 요약
users/orders를 로드해 `signup_year`(users), `order_date`(orders) 컬럼을 추가하고 `user_id` 기준 inner join. `StructType` 스키마로 타입 안정성 확보 후 `amount >= 100` 필터링.

### 정답 코드
```python
import logging
from pyspark.sql import SparkSession, functions as F
from pyspark.sql.types import (
    StructType, StructField, IntegerType, StringType, DateType, DoubleType,
)

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger("UserOrderETL")

AWS_ACCESS_KEY = ' '
AWS_SECRET_KEY = ' '
S3_USERS  = "s3a://<my-bucket>/userorder/users.csv"
S3_ORDERS = "s3a://<my-bucket>/userorder/orders.json"

spark = (
    SparkSession.builder
    .appName("user_order_dataset")
    .master("local[*]")
    .config("fs.s3a.access.key", AWS_ACCESS_KEY)
    .config("fs.s3a.secret.key", AWS_SECRET_KEY)
    .getOrCreate()
)

try:
    # 1) 로드
    users  = spark.read.csv(S3_USERS, header=True, inferSchema=True)
    orders = spark.read.option("multiLine", "true").json(S3_ORDERS)

    # 2) 파생 컬럼
    users  = users.withColumn(
        "signup_year",
        F.year(F.to_timestamp(F.col("signup_ts"), "yyyy.M.d H:m")),
    )
    orders = orders.withColumn("order_date", F.to_date(F.col("order_ts")))

    # 3) inner join + 컬럼 순서 지정
    joined = (
        users.join(orders, on="user_id", how="inner")
             .select("order_id", "user_id", "name", "gender",
                     "signup_year", "order_date", "amount")
    )

    # 4) StructType으로 타입 스키마 고정 (Dataset 역할)
    schema = StructType([
        StructField("order_id",    IntegerType(), True),
        StructField("user_id",     IntegerType(), True),
        StructField("name",        StringType(),  True),
        StructField("gender",      StringType(),  True),
        StructField("signup_year", IntegerType(), True),
        StructField("order_date",  DateType(),    True),
        StructField("amount",      DoubleType(),  True),
    ])
    joined_ds = spark.createDataFrame(joined.rdd, schema)

    # 5) amount >= 100 필터링
    filtered = joined_ds.filter(F.col("amount") >= 100)
    print("=== amount >= 100 주문 ===")
    filtered.show(truncate=False)

except Exception as e:
    logger.error(f"user-order ETL 실패: {e}", exc_info=True)
    raise
```

### 핵심 포인트
- users/orders DataFrame 로딩 O
- `signup_year` / `order_date` 컬럼 추가 O
- `user_id` inner join + 컬럼 순서 지정 O
- `StructType` 타입 스키마 정의 O
- `amount >= 100` 필터링 + 캡쳐본 O

## 문제 7. UDF 적용 + Spark SQL 비교

### 문제 요약
`signup_year`를 기준으로 `OLD`(~2019) / `MID`(2020~2022) / `NEW`(2023~)를 반환하는 UDF를 정의하고 `withColumn`으로 `user_segment` 컬럼 추가. 이어서 동일 로직을 Spark SQL `CASE WHEN`으로도 수행하여 비교.

### 정답 코드
```python
from pyspark.sql.functions import udf
from pyspark.sql.types import StringType

# (문제 6에서 이어짐: joined_ds 사용)

# 1) UDF 정의
def classify_segment(year: int) -> str:
    if year is None:
        return "UNKNOWN"
    if year < 2020:
        return "OLD"
    elif 2020 <= year <= 2022:
        return "MID"
    else:
        return "NEW"

segment_udf = udf(classify_segment, StringType())

# 2) DataFrame API + UDF
df_udf = joined_ds.withColumn("user_segment", segment_udf(F.col("signup_year")))
print("=== UDF 결과 ===")
df_udf.show(truncate=False)

# 3) Spark SQL로 동일 로직 비교
joined_ds.createOrReplaceTempView("joined")
# (또는 UDF를 SQL에 등록하려면: spark.udf.register("classify_segment", classify_segment, StringType()))
df_sql = spark.sql("""
    SELECT *,
           CASE
             WHEN signup_year < 2020 THEN 'OLD'
             WHEN signup_year BETWEEN 2020 AND 2022 THEN 'MID'
             ELSE 'NEW'
           END AS user_segment
    FROM joined
""")
print("=== Spark SQL 결과 ===")
df_sql.show(truncate=False)
```

### 핵심 포인트
- Python UDF 정의 + `udf()` 등록 O
- `withColumn("user_segment", ...)` 적용 O
- `createOrReplaceTempView` + `spark.sql(...)`로 동일 로직 비교 O
- UDF 결과 + SQL 결과 캡쳐본 제출 O

    "owner": "airflow",
    "depends_on_past": False,
    "email_on_failure": False,
    "email_on_retry": False,

    default_args=default_args,

    catchup=False,

        conn_id="spark_default",

## 문제 10. Temporary View + Spark SQL (JOIN + GROUP BY)

### 문제 요약
users/transactions DataFrame을 로드 → `createOrReplaceTempView`로 임시 뷰 등록 → Spark SQL로 `JOIN` + `GROUP BY`하여 사용자별 총 거래금액/건수를 집계.

### 정답 코드
```python
import logging
from pyspark.sql import SparkSession

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger("TempViewAgg")

AWS_ACCESS_KEY = ' '
AWS_SECRET_KEY = ' '
S3_USERS = "s3a://<my-bucket>/tempview/users.csv"
S3_TX    = "s3a://<my-bucket>/tempview/transactions.csv"

spark = (
    SparkSession.builder
    .appName("tempview_sql")
    .master("local[*]")
    .config("fs.s3a.access.key", AWS_ACCESS_KEY)
    .config("fs.s3a.secret.key", AWS_SECRET_KEY)
    .getOrCreate()
)

try:
    # 1) DataFrame 로딩
    users = spark.read.csv(S3_USERS, header=True, inferSchema=True)
    tx    = spark.read.csv(S3_TX,    header=True, inferSchema=True)
    logger.info("users / transactions 로드 완료")

    # 2) Temporary View 등록
    users.createOrReplaceTempView("users")
    tx.createOrReplaceTempView("transactions")
    logger.info("Temporary View 등록 완료")

    # 3) JOIN + GROUP BY
    result = spark.sql("""
        SELECT
            u.user_id,
            u.age,
            COUNT(t.transaction_id) AS tx_count,
            ROUND(SUM(t.amount), 2) AS total_amount
        FROM users u
        INNER JOIN transactions t
          ON u.user_id = t.user_id
        GROUP BY u.user_id, u.age
        ORDER BY total_amount DESC
    """)

    # 4) 실행 계획 확인(옵션) + 결과 출력
    result.explain(True)
    result.show(20, truncate=False)

except Exception as e:
    logger.error(f"TempView/SQL 실행 실패: {e}", exc_info=True)
    raise
```

### 핵심 포인트
- users/transactions DataFrame 생성 또는 로딩 O
- `createOrReplaceTempView("users")`, `createOrReplaceTempView("transactions")` O
- Spark SQL `INNER JOIN` + `GROUP BY` 수행 O
- 사용자별 합계(`SUM`) / 건수(`COUNT`) 집계 O
- 실행 결과 캡쳐본 제출 O

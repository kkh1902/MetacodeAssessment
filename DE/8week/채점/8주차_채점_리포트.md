## 5. 핵심 인사이트

### 가장 많이 틀린 과제: 과제10 (Kubernetes) - 63% 오답

- **12명 X** 중 대부분이 HPA 관련 자료(YAML 또는 스크린샷) 미제출
- Deployment+Service는 대부분 작성했으나, HPA까지 완성한 학생은 7명뿐
- HPA 개념은 이해했으나 실습 환경(EKS 비용, Metrics Server 등) 문제로 미완성 추정

### 두 번째: 과제4 (Airflow-Spark 연동) - 26% 오답

- **5명 X**: spark-submit 대신 PythonOperator+subprocess 또는 직접 PySpark 실행
- SparkSubmitOperator 사용법이 익숙하지 않은 학생들이 우회 방식 선택

### 특이사항

- **박지민(0점)**: docx에 코드가 전부 이미지로 삽입 → 텍스트 추출 불가. 직접 이미지 확인 시 점수 변동 가능

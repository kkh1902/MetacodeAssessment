## 선택 가이드

| 상황 | 추천 안 |
|---|---|
| **시간/비용 최소로 LLM 저항만 올리고 싶다** | **A** — 스킬·답지 구조 유지하며 디테일 추가 |
| **포켓몬 테마를 활용한 재미 있는 난이도 상승** | **B** — 학생 몰입도 ↑, 임의 규칙으로 LLM 혼란 |
| **SQL 비중을 실무 수준으로 끌어올리고 싶다** | **C** — 서브쿼리/윈도우/트랜잭션 실습, 취업 대비 |
| **커리큘럼 자체를 개편하며 새 스킬 도입** | **D** — 시각화·DB 성능까지 확장, 분량·리뷰 시간 가장 큼 |

**복합 선택도 가능**: 예컨대 A의 Q2 + C의 Q1 같은 혼합도 고려 가능. 각 md 문서 "타 안과 혼용 시" 섹션 참고.

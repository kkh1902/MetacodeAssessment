## Q4. 함수 선언 `prep_function` (10점)

**요구사항**
- `prep_function(df, drop_col, drop_data)` 선언
- before/after count 비교 + `pct_change` (소수 1자리)
- `before == 0`이면 `np.nan`
- 제외된 Type도 **after=0으로 반환 필수** (행 유지)
- 반환은 단일 DataFrame
- **별도 셀**에서 결과로 grouped bar chart 출력

**O**: 함수 반환 DataFrame + after=0 유지 + pct_change 포맷 + 그래프 제목
**X**: 제외된 Type 행 삭제 / pct_change 소수 포맷 오류 / 함수 외부에서 그래프 그리지 않음
